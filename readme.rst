=================
s265 HEVC Encoder
=================

| **Read:** | Online `documentation <http://s265.readthedocs.org/en/master/>`_ | Developer `wiki <http://bitbucket.org/multicoreware/s265_git/wiki/>`_
| **Download:** | `releases <http://ftp.videolan.org/pub/videolan/s265/>`_ 
| **Interact:** | #s265 on freenode.irc.net | `s265-devel@videolan.org <http://mailman.videolan.org/listinfo/s265-devel>`_ | `Report an issue <https://bitbucket.org/multicoreware/s265/issues?status=new&status=open>`_

`s265 <https://www.videolan.org/developers/s265.html>`_ is an open
source HEVC encoder. See the developer wiki for instructions for
downloading and building the source.
