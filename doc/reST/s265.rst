s265 CLI Documentation
######################


SYNOPSIS
========

**s265** [options] infile [-o] outfile

Bit depth: 8


**s265-10bit** [options] infile [-o] outfile

Bit depth: 10


infile can be YUV or Y4M

outfile is raw HEVC bitstream
